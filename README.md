# Modal Vue

## Описание

Приложение контроля показа модальных окон. Позволяет создавать и упорядочивать цепочки модальных окон.

## Стек

- Vue3
- Typescript
- SCSS

## Установка

- Склонируйте репозиторий `git clone git@github.com:Famelbyby/modal-vue.git `
- Перейдите в папку проекта `cd modal-vue`
- Установите зависимости `npm i`
- Запустите в dev-режиме `npm run dev` или соберите проект `npm run build && npm run preview`

## Опционально

- Отформатировать `npm run format`
- Проверить линтером `npm run lint`
